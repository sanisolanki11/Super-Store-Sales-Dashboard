# 📊 Super Store Sales Dashboard

## 📌 Project Overview

This project presents an interactive **Sales Dashboard** built using Power BI to analyze business performance across different regions, categories, and time periods.

The dashboard helps in understanding sales trends, profit distribution, and customer behavior to support data-driven decision-making.

---

## 🚀 Key Metrics

* 💰 Total Sales: 1.57M
* 📦 Total Quantity: 22K
* 📈 Total Profit: 175.26K
* 📊 Average Sales: 9.57

---

## 📊 Dashboard Features

* Sales by Payment Mode (COD, Online, Cards)
* Sales by Region (West, East, Central, South)
* Monthly Sales & Profit Trends (2019 vs 2020)
* Sales by Ship Mode
* Sales by Category & Sub-Category
* Profit & Sales by State (Map Visualization)

---

## 🔍 Key Insights

* COD is the most preferred payment mode (~43%)
* West region contributes the highest sales (~33%)
* Phones are the top-performing sub-category
* Profit shows fluctuations with a dip around April
* Standard Class shipping is the most used
